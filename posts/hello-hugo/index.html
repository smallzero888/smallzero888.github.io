Hello Hugo！我的第一篇技术博客#

🌱 这是我（smallzero）用 Hugo + PaperMod 搭建的第一篇博客，未来将聚焦 编程技巧、工具探索、创意实践。

为什么选择 Hugo？#

🚀 毫秒级构建速度#

Hugo 基于 Go 开发，即使管理 数百篇文章，也能瞬间完成站点生成。相比其他静态生成器，这种效率让我可以更专注于内容创作，无需等待漫长的构建过程。

🔒 静态站点的安全性#

静态站点不依赖数据库，天然避免 SQL 注入、数据库漏洞等风险。配合 GitHub Pages 部署，既能享受免费托管，又能通过 HTTPS 保障访问安全。#

🎨 PaperMod 主题的优雅体验#

选择 PaperMod 的核心原因：

  • 自动跟随系统的 深色/浅色模式 切换（读者体验友好）
  • 内置 代码高亮、阅读时间统计、分享按钮 等实用功能
  • 极简设计风格，让读者聚焦内容本身
